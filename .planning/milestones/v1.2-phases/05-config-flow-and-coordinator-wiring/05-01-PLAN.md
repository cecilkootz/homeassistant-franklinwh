## Task 02: Activate Modbus Fields in Config Flow

**File:** `/Users/matt/Development/homeassistant-franklinwh/custom_components/franklin_wh/config_flow.py`

**Changes:**
1. Import new constants: `CONF_LOCAL_PORT`, `CONF_LOCAL_SLAVE_ID`, `DEFAULT_LOCAL_PORT`, `DEFAULT_LOCAL_SLAVE_ID`
2. In `async_step_user()`, uncomment and complete Modbus fields in data_schema with:
   - `vol.Optional(CONF_USE_LOCAL_API, default=False): cv.boolean`
   - `vol.Optional(CONF_LOCAL_HOST): cv.string`
   - `vol.Optional(CONF_LOCAL_PORT, default=502): vol.All(vol.Coerce(int), vol.Range(min=1, max=65535))`
   - `vol.Optional(CONF_LOCAL_SLAVE_ID, default=1): vol.All(vol.Coerce(int), vol.Range(min=1, max=255))`
3. In `async_step_init()` (options flow), uncomment and complete Modbus fields:

   - Add host field: `vol.Optional(CONF_LOCAL_HOST): cv.string`
   - Add port field with validation: `vol.Optional(CONF_LOCAL_PORT, default=502): vol.All(vol.Coerce(int), vol.Range(min=1, max=65535))`

**Implementation Notes:**

- Default values: port=502, slave_id=1
- All Modbus fields are optional to allow cloud-only setup
